This project is a full stack E-Learning Notes application

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MySQL (XAMPP)

## Features
- User Authentication
- Create & Edit Notes
- Export Notes as PDF
- Modern UI with React

## How to Run Locally
1. Start MySQL (XAMPP)
2. Run backend:
   cd backend
   npm install
   node index.js
3. Run frontend:
   cd frontend-react
   npm install
   npm start
